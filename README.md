# Sales Dashboard - Power BI

## Overview
This project was completed as the final project of the NTI Data Analysis Training.

## Project Workflow
- Data Cleaning using Python (Pandas)
- Data Modeling (Star Schema)
- Power BI Dashboard Development
- DAX Measures
- Interactive Visualizations

## Features
- KPI Cards
- Slicers
- Tooltips
- Drill Down
- Bookmarks
- Filters

## KPIs
- Total Sales
- Total Orders
- Average Order Value
- Products Sold
- Total Customers
- Average Shipping Days

## Business Insights
- Consumer segment generates the highest sales.
- Technology is the top-selling category.
- Sales increased consistently from 2015 to 2018.
- Standard Class is the most used shipping mode.
- West region has the highest number of orders.

## Tools
- Power BI
- Python
- Pandas
- DAX
- Power Query
